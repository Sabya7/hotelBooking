1Ô∏è‚É£ SYSTEM OVERVIEW

The Hotel Management System (HMS) is a backend application that manages hotel operations including reservations, room inventory, billing, housekeeping, payments, reporting, staff operations, and integrations.

The system exposes REST APIs (JSON) and optionally WebSocket events for real-time updates.

Primary consumers:

Front Desk Web App

Housekeeping Mobile App

POS Systems

Channel Manager (Booking.com, Expedia, Airbnb, etc.)

Corporate admin portal
